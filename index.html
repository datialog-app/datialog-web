Requisitos del sistema

¿Funciona en mi ordenador?

Datialog está diseñado para funcionar en ordenadores estándar. No necesitas un PC gaming ni GPU dedicada para empezar.

Mínimo (Ollama local)
  • 💻 Sistema operativo: Windows 10/11, macOS 12+, Ubuntu 20+
  • 🧠 RAM: 8 GB mínimo (16 GB recomendado)
  • 💾 Disco: 10 GB libres (modelo IA ~2-4 GB)
  • ⚡ CPU: Intel Core i5 / AMD Ryzen 5 o superior
  • 🌐 Internet: Solo para instalación y activación de licencia
⏱ Con esta configuración los modelos responden en 10-30 segundos. Para mayor velocidad usa Groq (gratuito, online).
Recomendado (máximo rendimiento)
  • 💻 Sistema operativo: Windows 11, macOS 13+
  • 🧠 RAM: 16-32 GB
  • 💾 Disco: SSD con 20 GB libres
  • ⚡ CPU: Intel Core i7/i9, AMD Ryzen 7/9, Apple M1+
  • 🎮 GPU (opcional): NVIDIA 8GB+ VRAM para máxima velocidad
⚡ Con esta configuración los modelos responden en 2-5 segundos en local.
💡
¿Ordenador antiguo?

Puedes usar Datialog con Groq (gratuito) o las APIs de OpenAI y Anthropic sin instalar Ollama ni descargar modelos. Solo necesitas una conexión a internet y tu clave de API. Las respuestas son instantáneas.

Aviso legal: Datialog es una herramienta de apoyo al análisis de datos. El autor no se responsabiliza de los resultados obtenidos ni de posibles errores generados por los modelos de IA. El usuario es el único responsable de verificar los resultados antes de tomar decisiones importantes.

Privacidad: Con modelos locales (Ollama) los datos no salen de tu dispositivo. Al usar APIs externas (OpenAI, Anthropic, Groq) las consultas y el contexto de los datos se envían a sus servidores — el usuario asume esta responsabilidad y debe revisar sus políticas de privacidad antes de usar datos sensibles.
